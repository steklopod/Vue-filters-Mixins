# vue-cli

> Vue.js проект

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```

### Чтобы убить процесс на порту 8080 (Windows):
```
1. netstat -ano | findstr 8080
2. taskkill /pid @НОМЕР_ПОРТА@ /F
```
